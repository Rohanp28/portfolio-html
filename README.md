# Portfolio - HTML/CSS/JavaScript Version

This is a pure HTML, CSS, and JavaScript version of the React portfolio. It maintains the exact same design and functionality as the React version.

## Features

- ✅ Glassmorphic design with dark/light theme toggle
- ✅ Fully responsive layout
- ✅ Smooth scrolling navigation
- ✅ Mobile hamburger menu
- ✅ Scroll to top button
- ✅ Dynamic content rendering
- ✅ All sections: Hero, About, Skills, Experience, Projects, Services, Education, Contact

## File Structure

```
portfolio-html/
├── index.html      # Main HTML file
├── styles.css      # All CSS styles
├── script.js       # JavaScript functionality
├── resume.pdf      # Resume file (if available)
└── README.md       # This file
```

## How to Use

1. Simply open `index.html` in a web browser
2. No build process or dependencies required
3. All functionality works out of the box

## Features Implemented

- **Theme Toggle**: Switch between dark and light themes (persisted in localStorage)
- **Navigation**: Smooth scroll to sections
- **Mobile Menu**: Hamburger menu for mobile devices
- **Scroll to Top**: Button appears after scrolling 300px
- **Dynamic Content**: Technologies, experiences, and projects are rendered dynamically
- **Tech Icons**: All technology icons included as inline SVG

## Browser Support

Works in all modern browsers that support:
- CSS Grid
- Flexbox
- CSS Custom Properties (Variables)
- ES6 JavaScript

## Notes

- The resume file should be placed in the same directory as `index.html`
- All external links (LinkedIn, GitHub) open in new tabs
- Theme preference is saved in localStorage
